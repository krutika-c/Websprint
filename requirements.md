# WebSprint

## Project Overview
WebSprint is an interactive web development learning platform that helps users learn HTML, CSS, and JavaScript through lessons, quizzes, challenges, and level-based progression.

## Game Flow

### 1. Home Page
- Display the WebSprint logo and title.
- Show a "Start Game" button.

### 2. Assessment Test Page
- Ask users if they want to start the assessment.
- Include a "Start Test" button.

### 3. Initial Assessment Test
- Present questions on HTML, CSS, and JavaScript.
- Evaluate the user's knowledge level.

### 4. Technology Selection Page
- HTML
- CSS
- JavaScript

### 5. Level Selection Page
- Beginner
- Intermediate
- Advanced

### 6. Learning and Quiz Page
- Display topic explanations and examples.
- Provide quiz questions.
- Track lives and score.
- Users must complete the current level before progressing.

### 7. Level Completion Page
- Display score.
- Show concepts learned.
- Provide a "Next Level" button.

### 8. Final Completion Page
- Display final score.
- Show overall learning summary.
- Display:
  "Congratulations! You have completed WebSprint."
